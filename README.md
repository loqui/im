LOQUI Instant Messenger
=====
Loqui Instant Messenger unifies all your chat accounts in just one Firefox OS app
https://loqui.im

Providers support
===
At the moment it supports accounts from:
 * WhatsApp
 * Google Hangouts
 * Nimbuzz
 * Nokia ovi
 * Microsoft Lync
 * Any other XMPP/Jabber provider in the world
